# China A-Share Market Dashboard

This project is an interactive financial analytics dashboard for China's A-share market.

## Features

- Market overview
- Real-time A-share snapshot
- Custom sector analysis
- Cash flow assets analysis
- Core semiconductor analysis
- Semiconductor ecosystem analysis
- Smart money tracker
- Sector rotation comparison

## Data Sources

- BaoStock
- AKShare

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- BaoStock
- AKShare

## Project Purpose

This dashboard aims to analyze China's A-share market from the perspective of market performance, sector rotation, capital flow, and risk-return comparison.
